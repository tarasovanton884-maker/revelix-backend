# Revelix Backend Starter

Первый backend для Revelix.

## Что умеет
- `/health` — проверка, что сервер жив
- `/api/dashboard` — быстрые данные для Dashboard
- `/api/market-data` — расширенные данные для Market / Order Flow / Intelligence

## Локальный запуск
```bash
npm install
npm start
```

## Render
При создании Web Service:
- Environment: Node
- Build Command: `npm install`
- Start Command: `npm start`
- Instance Type: Starter
